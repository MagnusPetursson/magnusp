# magnusp
Repo for magnusp.is portfolio site. Made with Zensical. https://zensical.org/

## Portfolio Skeleton

This repo now includes a portfolio-focused content skeleton you can fill in incrementally.

### Main pages

- `docs/index.md` - Landing page
- `docs/about.md` - About section
- `docs/projects.md` - Project list and details
- `docs/experience.md` - Work experience
- `docs/skills.md` - Skills matrix
- `docs/contact.md` - Contact page
- `docs/templates.md` - Copy-paste content templates

Navigation is configured in `zensical.toml`.

## Local development

```bash
zensical serve
```

## Build

```bash
zensical build
```
